# 🏭 Vendor Performance analysis Dashboard

## 📌 Project Overview
Analyzed vendor performance data to identify key trends 
and built an interactive Power BI dashboard for 
management decision-making.
## 🛠️ Tools Used
- Python (Pandas, NumPy) — Data Cleaning
- SQL — Data Querying
-  Jupyter Notebook
- Power BI — Dashboard Development
## 📊 Key Results
- ✅ Cleaned 10,000+ rows of raw vendor data
- ✅ Reduced data inconsistencies by ~30%
- ✅ Analyzed performance trends across 15+ vendors
- ✅ Built 8-KPI interactive Power BI dashboard
- ✅ Cut vendor review time by 40%
- ✅ Improved on-time delivery tracking by 25%
